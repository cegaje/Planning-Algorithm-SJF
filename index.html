Algoritmo SJF
El algoritmo SJF es un algoritmo que se basa atender primero los procesos más cortos, es decir que requieran menos unidades de tiempo o el ciclo de CPU más pequeño (tiempo de ráfaga CPU), en caso haya una igualdad en el tiempo de menor ráfaga se aplicará el algoritmo FCFS (prioriza el primer proceso que llegó).
